# Gerenciador de Atendimentos - Clínica

Sistema completo de gerenciamento de atendimentos de clínica, integrando estruturas de dados clássicas em Python (Pilhas, Filas, Filas de Prioridade, Listas Encadeadas, Vetores Ordenados) e algoritmos eficientes (Busca Binária recursiva, Quicksort recursivo).

## Objetivo
Este projeto foi desenvolvido como um software modularizado, com tratamento robusto de erros, persistência e cobertura de testes.
